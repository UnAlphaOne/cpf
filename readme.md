## 📋 Description

**CPF (Codage Prédictif Fractal)** est un système d'intelligence artificielle auto-apprenante qui construit une base de connaissances dynamique à partir de multiples sources web. Il utilise un **algorithme fractal** pour organiser et connecter les concepts de manière hiérarchique et récursive.


 Intelligence Artificielle • Auto-apprentissage • Fractal  



## 🎯 Concept Fractal

Le système s'inspire des **fractales** - des structures qui se répètent à différentes échelles :

- **Niveau micro** : Mots-clés et concepts individuels
- **Niveau méso** : Relations et connexions entre concepts
- **Niveau macro** : Graphe global des connaissances
- **Niveau meta** : Auto-analyse et apprentissage récursif

Chaque concept est un "fragment fractal" qui contient en lui-même les graines d'autres concepts à explorer.

## ✨ Fonctionnalités Principales

### 🤖 Intelligence Auto-Apprenante
- Apprentissage continu depuis 20+ sources web
- Détection automatique des lacunes dans les connaissances
- Apprentissage récursif (approfondit chaque définition)

### 🧠 Algorithmes Avancés
- **Codage prédictif** : Anticipation des besoins de connaissance
